WORKFLOW: /delivery-checklist

Before delivery, produce this checklist and ensure all are covered:

- Build: command + expected success
- Run: how to start + expected behavior
- Key flows: 1–3 critical user paths verified
- Data: DB/migration impact + sanity check query/verification
- Regression: at least 1 previously working scenario verified
- Rollback: how to revert changes quickly
- Files changed: explicit list
