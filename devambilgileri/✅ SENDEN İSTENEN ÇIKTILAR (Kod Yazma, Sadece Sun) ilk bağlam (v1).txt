- Döngüyü `break` ile kırıyor.
- Gerçek virman **16.419,44 ₺** hiç kontrol edilmiyor.
- İptal satırı görmezden geliniyor.

### Domino Etkileri:
1. **Tutar Dengesi modülü:** 18.940,88 ₺'yi "fazla kayıt" olarak işaretliyor (sahte uyarı).
2. **Stopaj Uyuşmazlığı:** İptal edilen tutarı "Bankadan Çıkan"a dahil ediyor → `-18.940,88 ₺` sahte fark, kırmızı uyarı.
3. **Geçmiş/Açık Kayıtlar:** Yanlış virmanı listeliyor, açıklama boş görünüyor.

### Teknik Kök Neden:
Algoritma *"ilk eşleşen"* mantığıyla çalışıyor, *"iptal edilmemiş ilk eşleşen"* değil. Ayrıca aynı tutar birden fazla virmanla parçalı gönderildiğinde de `break` erken çıkıyor (ikinci bir bug).

## 🎯 ÇÖZÜM PLANI (Minimal Müdahale)

### Adım 1 — Yeni Fonksiyon: `iptal_edilenleri_isaretle()`

**Bağımsız, yeni bir fonksiyon.** Banka kayıtları karşılaştırmaya girmeden ÖNCE çağrılacak. Mevcut hiçbir kodu değiştirmez.

```python
def iptal_edilenleri_isaretle(banka_kayitlari):
    """
    Banka kayıtları içinde 'iptal' kelimesi geçen satırları bulur ve
    iptal edilen orijinal işlemle eşleştirir. Kayıtları SİLMEZ,
    sadece metadata ekler: iptal_edildi=True ve orijinal_referansi.
    """
    for i, kayit in enumerate(banka_kayitlari):
        islem_adi = str(kayit.get('islem_adi', '')).lower()
        
        if 'iptal' not in islem_adi:
            continue
        
        iptal_tutari = abs(float(kayit['islem_tutari']))
        iptal_zamani = kayit['islem_tarihi']
        
        # Orijinal işlemi bul: aynı tutar, zıt işaret, iptal'den önce
        for j, aday in enumerate(banka_kayitlari):
            if i == j:
                continue
            if aday.get('iptal_edildi'):
                continue
            if abs(float(aday['islem_tutari'])) != iptal_tutari:
                continue
            # İşaretler zıt olmalı (biri -, diğeri +)
            if (float(aday['islem_tutari']) * float(kayit['islem_tutari'])) >= 0:
                continue
            # Aday iptal'den ÖNCE olmalı
            if aday['islem_tarihi'] >= iptal_zamani:
                continue
            
            # EŞLEŞME
            aday['iptal_edildi'] = True
            aday['iptal_referansi'] = kayit.get('id') or j
            kayit['iptal_edildi'] = True
            kayit['orijinal_referansi'] = aday.get('id') or j
            break
    
    return banka_kayitlari
```

### Adım 2 — Mevcut Virman Döngüsüne TEK KOŞUL EKLE

Mevcut kod bloğunu BULUP sadece işaretli yere `continue` ekle. Başka hiçbir şeye dokunma.

```python
for kayit in banka_kayitlari:
    if kayit.get('tur') != 'Virman':
        continue
    
    # 👇 EKLENECEK BLOK 👇
    if kayit.get('iptal_edildi'):
        notlar.append(

            f"daha sonra iptal edilmiştir."
        )
        continue
    # 👆 EKLENECEK BLOK 👆
    
    # ... mevcut eşleştirme mantığı AYNEN kalır ...
```

### Adım 3 — Erken `break` Sorununu Düzelt (Parçalı Virman Desteği)

Aynı tutar birden fazla virmanla bölünerek gönderilebilir. Mevcut `break` ilk eşleşmede çıkıyor.

```python
# ÖNCE (hatalı):
if eslesme_bulundu:
    break

# SONRA (doğru):
toplam_eslesen = 0
eslesen_kayitlar = []
for kayit in uygun_virmanlar:
    if kayit.get('iptal_edildi'):
        continue
    toplam_eslesen += abs(float(kayit['tutar']))
    eslesen_kayitlar.append(kayit)
    if toplam_eslesen >= hedef_tutar:
        break  # sadece HEDEFE ulaşınca çık
```
